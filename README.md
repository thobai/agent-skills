# agent-skills

Reusable scripts for AI agent harnesses (Kiro, Claude Code, Codex, etc.).

## Skills

| Skill | Description |
|-------|-------------|
| [notify-me](./notify-me/) | Send a Google Chat message to the user and wait for their reply |

## Usage

Each skill has a `SKILL.md` describing its interface. Copy into your agent's skill directory or call directly.
